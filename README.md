# Application de Gestion de Caisse (Java)

## Description

Cette application de gestion de caisse a été développée en Java dans le but de gérer les opérations de vente dans un commerce.
Elle permet de gérer les produits, d’enregistrer les ventes et de structurer les données de manière claire et maintenable.

---


## Fonctionnalités

* Gestion des produits (ajout, modification, suppression)
* Gestion des ventes
* Calcul automatique des montants
* Recherche de produits
* Connexion à une base de données
* Interface utilisateur simple et intuitive

---

## Technologies utilisées

* Langage : Java
* Interface : JavaFX
* Base de données : MySQL
* Architecture : Programmation Orientée Objet (POO)
